HTML

HyperText Markup Language, more commonly known as HTML, is the langauge that is commonly used to create web pages. Web browsers such as Google Chrome use the HTML files in order to create these webpages that can be viewed by people all over the world. Thanks to HTML's method of describing its structure to a browser, it is considered a markup language as opposed to being considered a programming language.

In order to write HTML code, webpage creators must use these things called tags. In order to dictate that a tag is being used, the person writing the code must place their tag within angle brackets (<"tag">). For example, I used the tag "em" in that last sentence in order to italicize the word "tag". Tags have ability to dictate where headings, paragraphs, footers, lists, images, and links (among others)
